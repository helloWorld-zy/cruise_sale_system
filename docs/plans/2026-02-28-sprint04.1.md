## 前端设计规范（三端差异化设计）

> 三个前端产品面向不同用户群体和使用场景，**必须**采用差异化的视觉设计语言。但 Web 前台和小程序前台面向终端用户时，展示的**业务数据内容**（邮轮信息、舱位信息、价格、库存等）必须完全一致，仅在视觉风格和交互模式上有所区分。

### Admin 管理后台 — 专业数据驱动型设计

**设计理念：** 企业级 SaaS 管理控制台。追求高信息密度、高效操作、低视觉干扰的专业工具型风格。参考 Linear、Vercel Dashboard、Stripe Dashboard 的克制优雅。

**色彩体系：**
- 主色：`Indigo-600` (#4F46E5) — 用于主操作按钮、激活态、链接
- 背景色：`Slate-50` (#F8FAFC) 页面底色 / `White` 卡片/面板底色
- 边框色：`Slate-200` (#E2E8F0) — 用于卡片边框、分割线、表格边框
- 文字色：`Slate-900` (#0F172A) 标题 / `Slate-600` (#475569) 正文 / `Slate-400` (#94A3B8) 次要信息
- 状态色：`Emerald-500` 上架/成功 / `Amber-500` 维护中/警告 / `Rose-500` 下架/错误 / `Sky-500` 信息提示

**排版：**
- 字体：`Inter`（英文）、`Noto Sans SC`（中文），通过 Google Fonts 引入
- 标题：20px/600 → 16px/600 → 14px/600 三级递减
- 正文：14px/400，行高 1.6
- 表格/数据：13px/400 紧凑排列

**核心组件风格：**
- **表格**：斑马纹(`Slate-50`交替行)、悬浮高亮(`Indigo-50`)、固定表头、列宽可调节
- **筛选栏**：紧凑水平布局，搜索框 + 下拉筛选 + 操作按钮在同一行，背景白色卡片，`border-radius: 8px`
- **表单**：垂直标签布局，每个字段组间距 `16px`，标签 `Slate-600 13px/500`，输入框高度 `38px`，聚焦时 `ring-2 ring-Indigo-500`
- **状态标签**：圆角药丸 `rounded-full px-2.5 py-0.5 text-xs font-medium`，颜色对应状态色浅底+深文字（如上架=`bg-emerald-50 text-emerald-700`）
- **批量操作栏**：选中行后底部浮出操作条（`fixed bottom-0`），`bg-Indigo-600 text-white`，显示选中数量和可用操作
- **数据卡片（Dashboard）**：白底圆角卡片 `shadow-sm border`，内含标题、数值（`text-2xl font-bold`）、趋势箭头
- **富文本编辑器区域**：白底、`min-height: 300px`、工具栏 `bg-Slate-50` 置顶、`border rounded-lg`
- **图片上传区域**：虚线边框 `border-dashed border-2 border-Slate-300`，拖拽热区、缩略图网格 `gap-3`，主图角标 `bg-Indigo-600 text-white text-xs`

**动效：** 极简过渡 `transition-all duration-150`，仅用于 hover、focus、展开折叠。禁止花哨动画。
